This project is a **Streamlit web interface** for interacting with a locally installed Large Language Model (LLM) using **Ollama**. Users can chat with the AI, see answers in real time, and keep a conversation history.

## Features

- Connects to a locally installed LLM via Ollama.
- User-friendly chat interface with **Streamlit**.
- **Chat history panel** in the sidebar.
- **Reset button** to clear conversation.
- Real-time model responses.
